## AppImage with ELinks, and quickjs

Icon is created by 0x32f83b.

Build it with `nohup ./refresh-kruciai`

Now build usualy contains [elinks](https://github.com/rkd77/elinks ), [qjs and qjsc](https://github.com/quickjs-ng/quickjs ).

Also you can run bash session in it: `./$FILENAME.AppImage --bash`
